# RandomMog - The Ultimate Transmog Randomizer



Tired of spending hours in the transmog window trying to find the perfect look? Want to discover amazing combinations you never thought of? **RandomMog** adds a simple yet powerful randomization system to your transmog window that instantly creates new looks from your collected appearances!



Perfect for sharing laughs with friends on Discord as you discover the weird and wonderful outfits hiding in your collection!

## üéÆ How to Use



### Basic Usage

1. Open the transmog window at any transmog NPC or with the Grand Expedition Yak/Mammoth

2. Select which slots you want to randomize using the checkboxes

3. Click the **"Random!"** button at the bottom of the window (or **Shift-Click** for instant)

4. Watch your selected slots transform with smooth animation!

5. Don't like it? Click again for a completely new look!

6. Apply the transmog when you find something you like



### Slot Selection

- **Checkboxes** appear next to each equipment slot when transmog window is open

- **Weapon Enchants** have separate checkboxes next to the enchant slots

- **Split Shoulders** automatically show separate left/right checkboxes when enabled

- Use **"All"** button to select all slots

- Use **"None"** button to deselect all slots

- Your preferences are saved automatically!



### Advanced Features

- **Shift-Click** the Random button for instant randomization (no animation)

- **Weapon Enchants** - Check the enchant boxes to randomize illusions too

- **Split Shoulders** - Enable "Transmog Each Shoulder Separately" in WoW settings for independent shoulders

- **BetterWardrobe** - Both addons work together perfectly



### Slash Commands

- `/randommog` or `/rm` - Instantly randomize selected slots (no animation)

- Works whenever the transmog window is open



## üîß Installation



### Via CurseForge Client (Recommended)

1. Search for "RandomMog" in the CurseForge client

2. Click Install

3. Done! The addon loads automatically



### Manual Installation

1. Download RandomMog

2. Extract the RandomMog folder to your WoW AddOns directory:

   - Windows: `C:\Program Files (x86)\World of Warcraft\_retail_\Interface\AddOns\`

   - Mac: `/Applications/World of Warcraft/_retail_/Interface/AddOns/`

3. Restart WoW or type `/reload` in-game
